## 📊 按钮规格统一标准

### 高度标准

| 屏幕尺寸 | 主要操作按钮 | 次要操作按钮 | 控制按钮 |
| -------- | ------------ | ------------ | -------- |
| > 768px  | 默认         | 默认         | 默认     |
| ≤ 768px  | 48px         | 40px         | 36px     |
| ≤ 480px  | 44px         | 40px         | 36px     |

### 字体大小

| 按钮类型 | 768px | 480px |
| -------- | ----- | ----- |
| 主要操作 | 16px  | 15px  |
| 批量操作 | 14px  | 14px  |
| 订单操作 | 14px  | 14px  |
| 商品操作 | 13px  | 13px  |

### 间距规范

| 元素         | 间距        |
| ------------ | ----------- |
| 按钮之间     | 8px - 12px  |
| 按钮组内边距 | 12px - 16px |
| 分隔线距离   | 8px - 12px  |
